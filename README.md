# NodeJs package starter

Boilerplate to create NodeJs packages more easy.

## Usage

```bash
git clone https://github.com/jlopezxs/node-package-starter.git PACKAGENAME
cd PACKAGENAME
# Update the package.json with your data and
yarn install
```
